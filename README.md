## 给家里中老年辈不太会用电脑 写的小脚本
### 调整屏幕.bat
选择使用bat因为ps1一般被禁用，调整有点麻烦<br>
后续可以使用快捷方式来更改其在桌面的图标，让家里长辈在电脑上找起来比较明显方便<br>
  
这里是在屏幕缩放比例175%和200%中间变更，只要双击即可。<br>
每次更改屏幕大小，需要重新登录一下<br>
  
如果想要之后更改脚本中的屏幕大小 - LogPixels = 96 × Scaling Factor<br>
| 屏幕比例 | LogPixels  | Hex Value |
| ------- | ---------- | --------- |
| 100%    |  96        | 0x60      |
| 110%    |  106       | 0x6A      |
| 125%    |  120       | 0x78      |
| 135%    |  130       | 0x82      |
| 150%    |  144       | 0x90      |
| 160%    |  154       | 0x9A      |
| 175%    |  168       | 0xA8      |
| 180%    |  173       | 0xAD      |
| 190%    |  182       | 0xB6      |
| 200%    |  192       | 0xC0      |
